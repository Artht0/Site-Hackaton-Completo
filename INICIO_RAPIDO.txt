┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  ⚡ INSTALAÇÃO EM 3 PASSOS                                                  │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

  1️⃣  BACKEND
  
      cd backend
      npm install
      
      Crie arquivo .env:
      ┌─────────────────────────────────────┐
      │ PORT=5000                           │
      │ EMAIL_SERVICE=gmail                 │
      │ EMAIL_USER=seu@email.com            │
      │ EMAIL_PASS=senha-de-app             │
      │ FRONTEND_URL=http://localhost:3000  │
      └─────────────────────────────────────┘
      
      npm start
      
      ✅ Rodando em http://localhost:5000


  2️⃣  FRONTEND
  
      cd frontend
      npm install
      npm run dev
      
      ✅ Rodando em http://localhost:3000


  3️⃣  TESTAR
  
      → Abra http://localhost:3000
      → Clique "Inscreva-se Agora"
      → Preencha o formulário
      → Envie e verifique seu e-mail!


┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  🔑 SENHA DE APP DO GMAIL (IMPORTANTE!)                                     │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

  NÃO use sua senha normal do Gmail!
  
  ❌ Errado: sua senha de login normal
  ✅ Correto: senha de app (16 caracteres)
  
  Como conseguir:
  
  1. Acesse: https://myaccount.google.com/security
  2. Ative "Verificação em duas etapas"
  3. Procure "Senhas de app"
  4. Gere uma senha para "E-mail"
  5. Cole no arquivo .env
  
  Mais detalhes: CONFIGURACAO_EMAIL.md

┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  🐛 PROBLEMAS COMUNS                                                        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

  ❌ "E-mail não enviado"
     → Use senha de app do Gmail, não sua senha normal
     → Veja CONFIGURACAO_EMAIL.md
  
  ❌ "CORS Error"
     → Certifique-se de que o backend está rodando
     → Verifique se a porta está correta (5000)
  
  ❌ "Network Error"
     → Backend não está rodando
     → Execute: cd backend && npm start
  
  ❌ "npm: command not found"
     → Instale Node.js: https://nodejs.org/
