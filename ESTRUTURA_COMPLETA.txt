╔══════════════════════════════════════════════════════════════════════════════╗
║                                                                              ║
║                    📊  FLUXO DE DADOS                                        ║
║                                                                              ║
╚══════════════════════════════════════════════════════════════════════════════╝

  1. Usuário acessa frontend (Vercel / localhost:3000)
                    ↓
  2. Navega para página de inscrição
                    ↓
  3. Preenche formulário e submete
                    ↓
  4. Frontend envia POST para backend (Render / localhost:5000)
                    ↓
  5. Backend valida dados
                    ↓
  6. Backend salva em inscricoes.json
                    ↓
  7. Backend envia e-mail via Nodemailer
                    ↓
  8. Backend retorna sucesso para frontend
                    ↓
  9. Frontend exibe mensagem e redireciona
                    ↓
  10. Usuário recebe e-mail de confirmação ✅

  5. Rode o frontend: cd frontend && npm run dev
